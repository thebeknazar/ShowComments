  3. В нужное месте вставить :
  4. В comm.tpl файле доступны следующие теги :
    1) {text} - текст комментария
    2) [user_url] * текст * [/user_url] - обернет текст внутри в ссылку на профиль автора комментария
    3) [color] * текст * [/color] - обернет текст внутри в цвет группы к которой относится автор комментария
    4) {rating} - выведет рейтинг новости (в числе)
    5) {views} - выведет кол-во просмотров новости
    6) {user_url} - ссылка на автора
    7) {user_name} - просто ник автора
    8) {title} - укороченный заголовок новости
    9) {long_title} - полный заголовок новости
    10){author} - автор с ссылкой на профиль ( с модальным окном )
    11){full_link} - ссылка на комментарий ( сразу в новость )
    12){comm_num} - коллво комментариев
    13){date} - Дата (возможно работать как и с датой в краткой новости)
    14){foto} - ссылка на фото юзера
    15)[not-comm] любой текст [/not-comm] - выводит любой текст заключеный в эти теги, если комментариев не найдено
    16)[comm] любой текст [/comm] - выводит любой текст заключеный в эти теги, если комментарии найдены
    17){error} - выводит текст о том что комментариев нету
    18){title limit="x"} - Выводится урезанный до X количества символов, заголовок новости.
    19){text limit="x"} - Выводится урезанный до X количества символов, текст комментария.
    20) [xfvalue_x] - Значение дополнительного поля "x" с новости, где "x" название дополнительного поля
    21) [xfgiven_x]* текст *[/xfgiven_x] - Выводится текст внутри тегов, если поле не пустое, если поле не имеет значения, то текст просто вырезается. Где "x" название дополнительного поля с новости.
    22) [xfnotgiven_X]* текст *[/xfnotgiven_X] - Выводят текст указанный в них если дополнительное поле не было задано при заполнении SEO информации о доп поле, где "х" это имя дополнительного поля с новости
    23) [user_xf_x] - Значение дополнительного поля "x" с юзера, где "x" название дополнительного поля с профиля
    24) [user_xg_x]* текст *[/user_xg_x] - Выводится текст внутри тегов, если поле не пустое, если поле не имеет значения, то текст просто вырезается. Где "x" название дополнительного поля с профиля.
    25) [user_nxg_X]* текст *[/user_nxg_X] - Выводят текст указанный в них если дополнительное поле не было задано при заполнении SEO информации о доп поле, где "х" это имя дополнительного поля с профиля
    26) [rating] текст [/rating] - Выводят заключенный в них текст, только если рейтинг для новости включен и убирают содержимое, если он был отключен при добавлении новости.
    27) [rating-type-1] текст [/rating-type-1] - Выводят текст, заключенный в них, если в настройках скрипта включено использование первого типа рейтинга 'Оценка'.
    28) [rating-type-2] текст [/rating-type-2] - Выводят текст, заключенный в них, если в настройках скрипта включено использование второго типа рейтинга 'Только Нравится'.
    30) [rating-type-3] текст [/rating-type-3] - Выводят текст, заключенный в них, если в настройках скрипта включено использование третьего типа рейтинга 'Нравится' или 'Не нравится'.
